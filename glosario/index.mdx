## A

### Algoritmo:

Es una secuencia de pasos para llegar a un objetivo, una receta para cocinar una torta, eso es un algoritmo.

Los pasos del algoritmo dependen de un contexto preestablecido, en el caso de la torta hay que saber que es la harina.

---

## B

### Backend:

Es un servicio web al que se le puede enviar y pedir informacion pero no tiene una pagina o interfaz visible.

Cuando hacemos login en una web, los datos que ingresamos (email y contraseña) van al Backend para ser verificados y este nos responde si tenemos acceso o no. La respuesta la vamos a ver en el Frontend, la interfaz grafica de la web.

### Bootcamp:

Curso teorico practico orientado a una tecnologia especifica.

---

## C

### Crypto:

Hace referencia a las criptomonedas que utilizan las tecnologias de criptografia y blockchain para funcionar.

### CSS:

Cascading Style Sheets, en español «Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento.

```css
body {
  color: orange;
}
```

Su estructura posee un identificador `body`, una propiedad `color` y un valor `orange`.

### CTA:

Call to Action, en español «Llamada a la acción », es un término de marketing para cualquier diseño que busca generar una respuesta inmediata o fomentar una venta inmediata.

Suele estar representado por un botón grande y llamativo que invita a hacer clic.

## D

### Dev / Developer:

Desarrolladora o Desarrollador en ingles, persona que se dedica al desarollo de software.

## E

### Expresiones:

Código que al ser ejecutado resulta en un valor.

```jsx
"hello" + "world"; // produce 'hello world'
a === 20 && b === 30; // produce true o false basado en los valores de a y b
```

## F

### Figma:

Herramienta (también producto y servicio) para diseñar interfaces web o mobile.

[Web de Figma](https://www.figma.com/)

### Flask:

Es un framework para desarollo web en Python.

[Web de Flask](http://flask.pocoo.org/)

### Footer:

Pie de página en ingles, hace referencia a la seccion que se encuentra al final de diseño de una web, suele contener la informacion de contacto y links útiles.

### Fronted:

En una web el Frontend hace referencia a la interfaz e interaccion del usuario.

Desde un punto de vista más técnico, es todo el codigo `html`, `css` y `javascript` que se ejecuta en el navegador del usuario que visita la web.

### Fullstack:

Se le dice asi al desarollador o desarrolladora que se dedica al Frontend y Backend en conjunto.

## G

### Git:

Es un software de control de versiones usado para administrar los cambios que se hacen a lo largo del tiempo en un proyecto (repositorio). Es decir, crea un historial de cambios en el proyecto.

Permite ver quien hizo cambios en un rachivo de codigo (o texto), cuando los hizo y cuales fueron esos cambios.

### GitHub:

Es una web que te permite crear repositorios para guardar en la nube tus proyectos usando Git.

---

## H

### HackerRank:

Es una empresa de tecnología que se enfoca en desafíos de programación competitiva, donde los desarrolladores compiten tratando de resolver los ejercicios propuestos usando las especificaciones proporcionadas.

[Web de HackerRank](https://www.hackerrank.com/)

### HTML:

De las siglas en inglés de HyperText Markup Language (lenguaje de marcado de hipertexto), hace referencia al lenguaje de marcado usado para la elaboración de páginas web.

```html
<div>
  <h1>Hola</h1>
  <div></div>
</div>
```

Su estructura posee un tags que van entre `<>` como `<div>` y `<h1>`, cada tag tiene que abrirse `<h1>` y cerrarse `</h1>` agregandole `/`. Entre el tag de apertura y el de cierre se puede poner contenido como texto u otros tags.

## M

### MySQL:

Es un sistema de gestión de bases de datos relacional. O en otras palabras es un programa que permite crear bases de datos para almacenar informacion.

Esta basada en el modelo entidad-relacion (por esto es "relacional") y guarda la informacion en tablas de datos, siendo esta accesible y editable utilizando el lenguaje SQL.
